## 1. 문제 풀이

$4$ 바이트이면 `long int`, $12$ 바이트면 `long long long int`와 같은 형식으로 출력하는 문제로 $N$ 바이트의 경우 `long`의 출력 횟수가 $N / 4$ 가 됨을 알 수 있다.

---

## 2. 코드

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();

        int N = Integer.parseInt(br.readLine());

        for (int i = 0; i < N / 4; i++) {
            sb.append("long ");
        }
        sb.append("int");

        System.out.println(sb);
    }
}
```

### 2. 풀이 [Java]

`String`의 `repeat` 메서드를 사용하면 좀 더 간단하게 작성할 수 있다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

        int N = Integer.parseInt(br.readLine());

        bw.write("long ".repeat(N / 4) + "int");
        bw.flush();
    }
}
```

### 3. 풀이 [C++]

```c++
#include <bits/stdc++.h>
```
