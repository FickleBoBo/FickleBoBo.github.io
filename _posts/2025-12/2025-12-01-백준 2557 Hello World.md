## 1. 문제 풀이

`Hello World!` 를 출력하는 문제로 출력 스트림을 활용해 출력만 하면 된다.

---

## 2. 코드

### 1. 구현 [Java]

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```

### 2. 구현 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cout << "Hello World!";
}
```
