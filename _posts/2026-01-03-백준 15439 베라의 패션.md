## 1. 문제 풀이

<br>

상의와 하의가 각각 $N$ 개씩 있으므로 모든 조합은 $N^2$ 가지이며, 같은 색상이 한 세트씩 있으므로 같은 색상 조합은 $N$ 가지가 있어서 서로 다른 색상 조합은 $N^2 - N$ 가지가 있다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());
        System.out.println(N * N - N);
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    cout << n * n - n;
}
```
