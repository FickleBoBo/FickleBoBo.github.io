## 1. 문제 풀이

<br>

임의의 타일링에서 오른쪽 끝이 1×2 타일로 끝나는 경우는 해당 타일보다 가로 길이가 $1$ 만큼 작은 타일링들에서 해당 타일을 붙이면 되고, 2×1 타일로 끝나는 경우는 해당 타일보다 가로 길이가 $2$ 만큼 작은 타일링들에서 해당 타일을 붙이면 된다. 따라서 $a_n = a_{n-2} + a_{n-1}$ 로 점화식을 세워 다이나믹 프로그래밍으로 해결했다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
class Solution {

    static final int MOD = 1_000_000_007;

    public int solution(int n) {
        if (n == 1) return 1;

        int[] dp = new int[1 + n];
        dp[1] = 1;
        dp[2] = 2;

        for (int i = 3; i <= n; i++) {
            dp[i] = (dp[i - 2] + dp[i - 1]) % MOD;
        }

        return dp[n];
    }
}
```
