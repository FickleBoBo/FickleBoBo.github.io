## 1. 문제 풀이

<br>

예제 입력이 $2541$ 일 때, 예제 출력이 $1998$ 이므로 서기 연도는 불기 연도에서 $2541 - 1998 = 543$ 을 빼면 구할 수 있는 것을 알 수 있다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int y = Integer.parseInt(br.readLine());
        System.out.println(y - 543);
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int y;
    cin >> y;
    cout << y - 543;
}
```
