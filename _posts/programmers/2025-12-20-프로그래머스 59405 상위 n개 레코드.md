## 1. 문제 풀이

<br>

보호 시작일을 기준으로 오름차순으로 정렬 후 `LIMIT`으로 가장 첫 레코드를 조회하면 된다. 또는 집계 함수와 서브쿼리로 보호 시작일이 가장 빠른 날을 찾고 이와 보호 시작일이 동일한 동물의 이름을 찾아도 된다.

---

## 2. 쿼리

<br>

### 1. ORDER BY + LIMIT

```sql
SELECT NAME
FROM ANIMAL_INS
ORDER BY DATETIME
LIMIT 1;
```

<br>

### 2. 집계 함수 + 서브쿼리

```sql
SELECT NAME
FROM ANIMAL_INS
WHERE DATETIME = (SELECT MIN(DATETIME) FROM ANIMAL_INS);
```
