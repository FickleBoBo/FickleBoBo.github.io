## 1. 문제 풀이

<br>

오늘이 일요일일 때, $X$ 일 후가 화요일인지 판단하는 문제로 $7$ 일 단위로 다시 일요일이 되므로 $X$ 를 $7$ 로 나눈 나머지가 $2$ 면 화요일이다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int x = Integer.parseInt(br.readLine());

        if (x % 7 == 2) {
            System.out.println(1);
        } else {
            System.out.println(0);
        }
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int x;

}
```
