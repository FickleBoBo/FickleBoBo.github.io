## 1. 아이디어

<br>

주어진 `MenOfPassion` 알고리즘은 파리미터로 넘어온 배열과 n에 대해 n을 2로 나눈 바닥값의 인덱스로 배열의 값을 반환한다. 입력의 크기와 무관하게 반환은 항상 1번만하므로 수행횟수는 1회이며, 시간 복잡도는 O(1)이라 최고차항의 계수는 0이다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
public class Main {
    public static void main(String[] args) {
        System.out.println(1);
        System.out.println(0);
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    cout << 1 << '\n';
    cout << 0 << '\n';
}
```
