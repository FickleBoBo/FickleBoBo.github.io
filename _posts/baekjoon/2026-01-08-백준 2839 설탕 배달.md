## 1. 문제 풀이

<br>

$3$ 킬로그램 봉지와 $5$ 킬로그램 봉지로 $N$ 킬로그램을 맞춰야 하는 문제로 사용한 $3$ 킬로그램 봉지가 $5$ 개를 넘어가면 그 $5$ 개 대신 $5$ 킬로그램 봉지 $3$ 개를 써서 더 적은 수의 봉지로 $N$ 킬로그램을 맞출 수 있다. 따라서 $3$ 킬로그램 봉지를 하나도 안썼을 때부터 $4$ 개를 썼을 때까지 중 $N$ 킬로가 존재하는지 찾으면 된다.

- $3$ 킬로그램 봉지를 아예 안썼을 때는 $N$ 을 $5$ 로 나눈 나머지가 $0$ 이면 $N$ 킬로그램을 맞출 수 있는 것이다.
- $3$ 킬로그램 봉지를 한 개 썼을 때는 $N - 3$ 을 $5$ 로 나눈 나머지가 $0$ 이면 $N$ 킬로그램을 맞출 수 있는 것이다.
- $3$ 킬로그램 봉지를 두 개 썼을 때는 $N - 6$ 을 $5$ 로 나눈 나머지가 $0$ 이면 $N$ 킬로그램을 맞출 수 있는 것이다.
- $3$ 킬로그램 봉지를 세 개 썼을 때는 $N - 9$ 을 $5$ 로 나눈 나머지가 $0$ 이면 $N$ 킬로그램을 맞출 수 있는 것이다.
- $3$ 킬로그램 봉지를 네 개 썼을 때는 $N - 12$ 을 $5$ 로 나눈 나머지가 $0$ 이면 $N$ 킬로그램을 맞출 수 있는 것이다.

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());

        if (N >= 0 && N % 5 == 0) {
            System.out.println(N / 5);
        } else if (N >= 3 && (N - 3) % 5 == 0) {
            System.out.println((N - 3) / 5 + 1);
        } else if (N >= 6 && (N - 6) % 5 == 0) {
            System.out.println((N - 6) / 5 + 2);
        } else if (N >= 9 && (N - 9) % 5 == 0) {
            System.out.println((N - 9) / 5 + 3);
        } else if (N >= 12 && (N - 12) % 5 == 0) {
            System.out.println((N - 12) / 5 + 4);
        } else {
            System.out.println(-1);
        }
    }
}
```
