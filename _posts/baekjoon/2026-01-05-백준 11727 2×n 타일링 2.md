## 1. 문제 풀이

<br>

임의의 타일링에서 오른쪽 끝이 1×2 타일로 끝나는 경우는 해당 타일보다 가로 길이가 $1$ 만큼 작은 타일링들에서 해당 타일을 붙이면 되고, 2×1 타일로 끝나는 경우는 해당 타일보다 가로 길이가 $2$ 만큼 작은 타일링들에서 해당 타일을 붙이면 되고, 2×2 타일로 끝나는 경우는 해당 타일보다 가로 길이가 $2$ 만큼 작은 타일링들에서 해당 타일을 붙이면 된다. 따라서 $a_n = 2 \times a_{n-2} + a_{n-1}$ 로 점화식을 세워 다이나믹 프로그래밍으로 해결했다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {

    static final int MOD = 10007;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int n = Integer.parseInt(br.readLine());
        if (n == 1) {
            System.out.println(1);
            return;
        }

        int[] dp = new int[1 + n];
        dp[1] = 1;
        dp[2] = 3;

        for (int i = 3; i <= n; i++) {
            dp[i] = (2 * dp[i - 2] + dp[i - 1]) % MOD;
        }

        System.out.println(dp[n]);
    }
}
```

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

constexpr int MAXN = 1000;
constexpr int MOD = 10007;

int dp[1 + MAXN] = {0, 1, 3};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    for (int i = 3; i <= n; i++) {
        dp[i] = (2 * dp[i - 2] + dp[i - 1]) % MOD;
    }

    cout << dp[n];
}
```
