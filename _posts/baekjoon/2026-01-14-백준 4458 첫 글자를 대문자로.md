## 1. 문제 풀이

<br>

요구사항대로 첫 글자만 대문자로 바꿔서 출력하면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

`Character.toUpperCase` 메서드로 첫 글자를 대문자로 바꾸어서 출력하고 `substring` 메서드로 나머지 글자들을 출력해줬다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

        int N = Integer.parseInt(br.readLine());
        for (int i = 0; i < N; i++) {
            String str = br.readLine();

            bw.write(Character.toUpperCase(str.charAt(0)));
            bw.write(str.substring(1));
            bw.newLine();
        }

        bw.flush();
    }
}
```

<br>

### 2. 풀이 [C++]

문자열이 가변이므로 첫 글자를 대문자로 변경해주고 출력했다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    cin.ignore();

    for (int i = 0; i < n; i++) {
        string line;
        getline(cin, line);

        line[0] = toupper(line[0]);
        cout << line << '\n';
    }
}
```
