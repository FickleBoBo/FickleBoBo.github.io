## 문제 풀이

모든 가로수가 같은 간격이 되도록 심어야 하는 가로수의 최소 개수를 구하는 문제로 주어진 입력에서 가로수들의 간격들에 대한 최대 공약수를 간격으로 설정하면 같은 간격을 유지하면서 가로수를 최소로 심을 수 있다.

---

## 코드

### 1. 유클리드 호제법 [Java]

가로수가 최소 3개는 주어지므로 앞에서 3개의 가로수에 대한 두 간격의 최대 공약수를 먼저 구했고, 이후 등장하는 간격과 현재 구한 최대공약수와의 최대공약수를 반복적으로 구해서 간격을 찾았다.

필요한 가로수의 수는 가장 가까운 나무와 먼 나무의 간격을 최대공약수로 나눈 값 + 1개인데 이미 `N`개는 심어져 있으니 이를 뺀 수만큼 더 심어주면 된다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());

        int[] arr = new int[N];
        for (int i = 0; i < N; i++) {
            arr[i] = Integer.parseInt(br.readLine());
        }

        int gcd = gcd(arr[1] - arr[0], arr[2] - arr[1]);
        for (int i = 3; i < N; i++) {
            gcd = gcd(gcd, arr[i] - arr[i - 1]);
        }

        System.out.println((arr[N - 1] - arr[0]) / gcd + 1 - N);
    }

    private static int gcd(int a, int b) {
        if (b == 0) return a;
        return gcd(b, a % b);
    }
}
```
