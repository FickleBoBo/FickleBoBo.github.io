## 문제 풀이

문제 제목에서도 볼 수 있듯 전형적인 LCS 알고리즘을 활용해서 해결할 수 있는 문제다. 일반적인 해법인 다이나믹 프로그래밍을 활용하면 해결할 수 있는데 Bottom-Up 방식과 Top-Down 방식 두 가지로 해결했다.

다이나믹 프로그래밍을 활용한 LCS의 기본 점화식인

$$
DP[i][j] =
\begin{cases}
0 & \text{if } i = 0 \text{ or } j = 0 \\\\
DP[i-1][j-1] + 1 & \text{if } A[i] = B[j] \\\\
\max(DP[i-1][j],\ DP[i][j-1]) & \text{otherwise}
\end{cases}
$$

를 활용하는 것은 두 가지 방식 모두 동일하다.

### 1. Bottom-Up DP

Bottom-Up 방식의 경우 LCS를 계산할 2차원 dp 배열을 생성한 후 쭉 순회하며 조건에 맞춰 값을 갱신해주기만 하면 된다.

### 2. Top-Down DP

Top-Down 방식의 경우 일반적인 재귀만으로는 해결할 수 없어서 메모이제이션을 활용해야 하는데 이때의 메모이제이션 테이블이 dp 테이블과 동일한 모양이 된다. 갱신 여부는 방문 체크 배열을 별도로 만들어서 방문 체크와 메모 및 메모값 반환으로 구성했다.

## 코드

### 1. Bottom-Up DP

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        char[] str1 = br.readLine().toCharArray();
        char[] str2 = br.readLine().toCharArray();
        int N = str1.length;
        int M = str2.length;

        int[][] dp = new int[1 + N][1 + M];
        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= M; j++) {
                if (str1[i - 1] == str2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        System.out.println(dp[N][M]);
    }
}
```

### 2. Top-Down DP

```java
import java.io.*;

public class Main {

    private static int[][] dp;
    private static boolean[][] visited;

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        char[] str1 = br.readLine().toCharArray();
        char[] str2 = br.readLine().toCharArray();
        int N = str1.length;
        int M = str2.length;

        dp = new int[N][M];
        visited = new boolean[N][M];

        System.out.println(lcs(str1, N - 1, str2, M - 1));
    }

    private static int lcs(char[] str1, int len1, char[] str2, int len2) {
        if (len1 == -1 || len2 == -1) return 0;

        if (visited[len1][len2]) return dp[len1][len2];

        visited[len1][len2] = true;
        if (str1[len1] == str2[len2]) {
            return dp[len1][len2] = lcs(str1, len1 - 1, str2, len2 - 1) + 1;
        } else {
            return dp[len1][len2] = Math.max(lcs(str1, len1 - 1, str2, len2), lcs(str1, len1, str2, len2 - 1));
        }
    }
}
```
