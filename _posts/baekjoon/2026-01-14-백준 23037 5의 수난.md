## 1. 문제 풀이

<br>

주어진 입력을 문자열로 받고 각 자리 수의 다섯 제곱의 합을 구하면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        String str = br.readLine();
        int sum = 0;

        for (char c : str.toCharArray()) {
            int n = c - '0';
            sum += n * n * n * n * n;
        }

        System.out.println(sum);
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;

    int sum = 0;
    for (char c : s) {
        int x = c - '0';
        sum += x * x * x * x * x;
    }

    cout << sum;
}
```
