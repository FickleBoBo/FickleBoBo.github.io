## 1. 문제 풀이

두 정수 $A$, $B$ 의 나눗셈을 산술 연산자를 활용하여 구하면 된다. 이때 정수 타입간 나눗셈 연산은 나머지가 버림 처리 되기 때문에 실수 타입을 활용해야 한다. 또한 실제 정답과의 오차가 $10^{-9}$ 이하여야 해서 소수점 아래 9자리까지는 출력을 해야한다.

---

## 2. 코드

### 1. 구현 [Java]

$A$ 를 `double` 형으로 형변환해서 소수점까지 나눗셈 연산이 잘 될 수 있도록 했다.

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int A = Integer.parseInt(st.nextToken());
        int B = Integer.parseInt(st.nextToken());

        System.out.println((double) A / B);
    }
}
```

### 2. 구현 [C++]

`cout` 은 소수점 아래 6자리까지만 출력하는 것이 기본값이기 때문에 `fixed` 와 `setprecision(9)` 을 통해 9자리까지 출력하도록 해줬다. 나눗셈의 경우 $A$ 를 `double` 형으로 형변환해서 소수점까지 나눗셈 연산이 잘 될 수 있도록 했다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int a, b;
    cin >> a >> b;
    cout << fixed << setprecision(9) << (double)a / b;
}
```
