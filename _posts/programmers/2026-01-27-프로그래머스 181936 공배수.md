## 1. 문제 풀이

<br>

배수는 나누었을 때 나머지가 $0$ 인지 여부로 간단하게 구할 수 있다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
class Solution {
    public int solution(int number, int n, int m) {
        if (number % n == 0 && number % m == 0) return 1;
        return 0;
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <string>
#include <vector>

using namespace std;

int solution(int number, int n, int m) {
    return number % n == 0 && number % m == 0;
}
```
