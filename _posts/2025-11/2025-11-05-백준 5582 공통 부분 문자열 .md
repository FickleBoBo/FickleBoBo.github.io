## 문제 풀이

두 문자열의 가장 긴 공통 부분 문자열을 찾는 문제로 [LCS]({% post_url 2025-11/2025-11-16-LCS (Longest Common Substring) %}) 를 활용하면 해결할 수 있다. 최장 공통 부분 수열이 아니라 최장 공통 부분 문자열임에 주의해야 한다.

최댓값의 경우도 최장 공통 부분 수열은 마지막 인덱스에서 얻을 수 있지만 최장 공통 부분 문자열은 어디에 저장됐을지 알 수 없다.

---

## 코드

### 1. Bottom-Up dp [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        char[] str1 = br.readLine().toCharArray();
        char[] str2 = br.readLine().toCharArray();
        int N = str1.length;
        int M = str2.length;

        int[][] dp = new int[1 + N][1 + M];
        int max = 0;

        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= M; j++) {
                if (str1[i - 1] == str2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                    max = Math.max(max, dp[i][j]);
                }
            }
        }

        System.out.println(max);
    }
}
```
