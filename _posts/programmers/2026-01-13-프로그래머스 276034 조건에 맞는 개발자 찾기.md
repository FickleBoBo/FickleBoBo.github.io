## 1. 문제 풀이

<br>

서브쿼리로 `Python`이나 `C#` 스킬의 코드를 가져와서 비트 AND 연산을 해줬다.

---

## 2. 쿼리

<br>

```sql
SELECT ID, EMAIL, FIRST_NAME, LAST_NAME
FROM DEVELOPERS
WHERE
    SKILL_CODE & (SELECT CODE FROM SKILLCODES WHERE NAME = 'Python') > 0 OR
    SKILL_CODE & (SELECT CODE FROM SKILLCODES WHERE NAME = 'C#') > 0
ORDER BY ID;
```
