## 1. 아이디어

<br>

오늘 날짜를 양식에 맞춰 출력만 하면 된다. 한국 시간은 세계 표준시보다 9시간 빨라서 한국 시간으로 오전 9시 전까지는 하루 전 날짜를 출력해야 한다.

## 2. 코드

### 1. 풀이 [Java]

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("2026");
        System.out.println("01");
        System.out.println("09");
    }
}
```

#include <bits/stdc++.h>
