## 1. 문제 풀이

주어진 $N$ 의 자리수를 내림차순으로 정렬한 수를 출력하는 문제로 자리수에 대한 계산을 해야하기에 주어진 수를 문자열로 다루어서 역순 정렬을 구현해야 한다.

---

## 2. 코드

### 1. 정렬 [Java]

수 $N$ 을 `char` 배열로 변환 후 정렬하고 역순으로 출력했다. `char` 의 경우 숫자의 비교 순서가 정수랑 똑같아서 가능하다.

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringBuilder sb = new StringBuilder();

        char[] arr = br.readLine().toCharArray();
        Arrays.sort(arr);

        for (int i = arr.length - 1; i >= 0; i--) {
            sb.append(arr[i]);
        }

        bw.write(sb.toString());
        bw.flush();
    }
}
```

### 2. 정렬 [Java]

카운팅 배열에 각 자리수를 카운팅한 후 역순으로 출력했다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringBuilder sb = new StringBuilder();

        String[] input = br.readLine().split("");
        int[] cntArr = new int[10];

        for (String str : input) {
            cntArr[Integer.parseInt(str)]++;
        }

        for (int i = 9; i >= 0; i--) {
            sb.append(String.valueOf(i).repeat(cntArr[i]));
        }

        bw.write(sb.toString());
        bw.flush();
    }
}
```

### 3. 정렬 [C++]

`string` 자체를 역순으로 정렬했다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;

    sort(s.rbegin(), s.rend());
    cout << s;
}
```
