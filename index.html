개발

개발자가 되고 싶은 이유
- 능력만 있으면 혼자 가치 있는 서비스를 생산할 수 있을 것 같다
- 별 의미 없는 자격증 말고 실력으로 승부 보는 것 같아서 맘에 든다
- 나중에 나라가 어려워지면 외국으로 가면 될 것 같다
- 현직 친구들에게 물어보면 좋다고 했다
- 지금까지 개발 공부를 하는 과정이 재밌어서 적성에 잘 맞을 것 같다

개발 경험(2년 전)
- C와 python으로 알고리즘까지 가볍게 공부하고 '나도코딩' 유튜브 보면서 클론 코딩
- 토이 프로젝트로 테트리스 구현

SSAFY를 선택한 이유
- 비전공자가 개발자가 되려면 최소 1년은 준비해야 한다고 생각했다
- 백엔드 개발자를 희망하는데 커리큘럼이 백엔드에 적합하다고 판단했다
- 여러 교육들을 수강했던 친구들에게 물어봤을 때 학업 분위기가 잘 형성된 것 같았다

최종 목표
프리랜서 or 취미 개발자가 되는 것이 최종 목표로 이를 위해 열심히 실력을 갈고닦고 싶다
