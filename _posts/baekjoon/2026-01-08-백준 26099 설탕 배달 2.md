## 1. 문제 풀이

<br>

[BaekJoon 2839번 - 설탕 배달](/posts/baekjoon-2839) 문제에서 $N$ 의 크기만 커진 문제로 동일한 아이디어로 해결할 수 있다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        long N = Long.parseLong(br.readLine());

        if (N >= 0 && N % 5 == 0) {
            System.out.println(N / 5);
        } else if (N >= 3 && (N - 3) % 5 == 0) {
            System.out.println((N - 3) / 5 + 1);
        } else if (N >= 6 && (N - 6) % 5 == 0) {
            System.out.println((N - 6) / 5 + 2);
        } else if (N >= 9 && (N - 9) % 5 == 0) {
            System.out.println((N - 9) / 5 + 3);
        } else if (N >= 12 && (N - 12) % 5 == 0) {
            System.out.println((N - 12) / 5 + 4);
        } else {
            System.out.println(-1);
        }
    }
}
```
