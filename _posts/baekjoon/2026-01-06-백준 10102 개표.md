## 1. 문제 풀이

<br>

$A$ 와 $B$ 의 등장 횟수를 비교만 해주면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int a = 0;
        int b = 0;

        int V = Integer.parseInt(br.readLine());
        String str = br.readLine();

        for (int i = 0; i < V; i++) {
            if (str.charAt(i) == 'A') {
                a++;
            } else {
                b++;
            }
        }

        if (a > b) {
            System.out.println("A");
        } else if (a < b) {
            System.out.println("B");
        } else {
            System.out.println("Tie");
        }
    }
}
```

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int a = 0;
    int b = 0;

    int v;
    string s;
    cin >> v >> s;

    for (char c : s) {
        if (c == 'A') {
            a++;
        } else {
            b++;
        }
    }

    if (a > b) {
        cout << 'A';
    } else if (a < b) {
        cout << 'B';
    } else {
        cout << "Tie";
    }
}
```
