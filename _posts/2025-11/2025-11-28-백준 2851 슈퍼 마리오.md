## 문제 풀이

슈퍼 마리오가 얻은 점수가 100에 최대한 가깝게 하는 문제로 가까운 수가 2개면 큰 값을 선택해야하며, 버섯은 안먹은 순간부터 뒤에 나오는 버섯은 전부 먹을 수 없다. 그냥 조건문 기반으로 구현하는 방식으로 해결해도 되고 누적합을 활용해서 해결해도 된다.

---

## 코드

### 1. 구현 [Java]

현재 버섯을 먹어도 100점이 안되면 먹고, 현재 버섯을 먹었을 때 100점을 넘으면 먹는 경우와 안먹는 경우를 비교했다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int[] arr = new int[10];
        for (int i = 0; i < 10; i++) {
            arr[i] = Integer.parseInt(br.readLine());
        }

        int sum = 0;
        for (int i = 0; i < 10; i++) {
            if (sum + arr[i] <= 100) {
                sum += arr[i];
                continue;
            }

            if ((100 - sum) >= (sum + arr[i] - 100)) {
                sum += arr[i];
                continue;
            }

            break;
        }

        System.out.println(sum);
    }
}
```

### 2. 누적합 [Java]

누적합 배열을 먼저 만들고 이후 누적합 배열에서 100점과 가장 가까운 값을 찾았다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int[] arr = new int[10];
        for (int i = 0; i < 10; i++) {
            arr[i] = Integer.parseInt(br.readLine());
        }

        int[] prefixSum = new int[1 + 10];
        for (int i = 1; i <= 10; i++) {
            prefixSum[i] = prefixSum[i - 1] + arr[i - 1];
        }

        int ans = 0;
        int diff = 1_000;
        for (int i = 1; i <= 10; i++) {
            if (Math.abs(prefixSum[i] - 100) <= diff) {
                ans = prefixSum[i];
                diff = Math.abs(prefixSum[i] - 100);
            }
        }

        System.out.println(ans);
    }
}
```
