## 1. 문제 풀이

주어진 녹색거탑을 내려오는 경우의 수를 구하는 문제로 각 칸에서 아래로 갈 수 있는 칸은 왼쪽 오른쪽 두 개씩 존재해서 현재 단계까지 올 수 있던 경우의 수가 $n$ 이면 다음 단계까지 갈 수 있는 경우의 수는 $2 \times n$ 이 된다. 비트 시프트 연산자로 $2$ 의 거듭제곱을 구하는 형태로 해결했다.

---

## 2. 코드

### 1. 수학 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());
        System.out.println(1 << N);
    }
}
```

### 2. 수학 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    cout << (1 << n);
}
```
