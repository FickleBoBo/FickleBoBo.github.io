## 1. 문제 풀이

<br>

주어진 글자들을 세로로 읽는 문제로 2차원 배열을 활용하여 입력을 받고 이를 열 우선 순회로 탐색하면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

`char` 타입 배열의 초깃값이 `'\u0000'`인 점을 활용했다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();

        char[][] grid = new char[5][15];
        for (int i = 0; i < 5; i++) {
            String input = br.readLine();

            for (int j = 0; j < input.length(); j++) {
                grid[i][j] = input.charAt(j);
            }
        }

        for (int j = 0; j < 15; j++) {
            for (int i = 0; i < 5; i++) {
                if (grid[i][j] != '\u0000') sb.append(grid[i][j]);
            }
        }

        System.out.println(sb);
    }
}
```

<br>

### 2. 풀이 [C++]

`char` 타입 배열의 초깃값이 `'\0'`인 점을 활용했다.

```c++
#include <bits/stdc++.h>
using namespace std;

char grid[5][15];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    for (int i = 0; i < 5; i++) {
        cin >> grid[i];
    }

    for (int j = 0; j < 15; j++) {
        for (int i = 0; i < 5; i++) {
            if (grid[i][j] != '\0') cout << grid[i][j];
        }
    }
}
```
