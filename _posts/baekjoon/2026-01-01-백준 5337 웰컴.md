## 1. 문제 풀이

<br>

여러 특수문자로 이루어진 Welcome을 출력하는 문제로 이스케이프해야 하는 문자가 섞여 있음에 주의해야 한다.

---

## 2. 코드

 <br>

### 1. 풀이 [Java]

Welcome을 이루는 문자 중 이스케이프 처리가 필요한 문자로 역슬래시(`\`)가 있어서 앞에 역슬래시를 붙여 이스케이프 처리만 추가해줬다.

```java
public class Main {
    public static void main(String[] args) {
        System.out.println(".  .   .");
        System.out.println("|  | _ | _. _ ._ _  _");
        System.out.println("|/\\|(/.|(_.(_)[ | )(/.");
    }
}
```

<br>

### 2. 풀이 [C++]

Welcome을 이루는 문자 중 이스케이프 처리가 필요한 문자로 역슬래시(`\`)가 있어서 앞에 역슬래시를 붙여 이스케이프 처리만 추가해줬다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cout << ".  .   ." << '\n';
    cout << "|  | _ | _. _ ._ _  _" << '\n';
    cout << "|/\\|(/.|(_.(_)[ | )(/." << '\n';
}
```
