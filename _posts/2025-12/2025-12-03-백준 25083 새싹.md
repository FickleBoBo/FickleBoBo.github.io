## 1. 문제 풀이

여러 특수문자로 이루어진 새싹을 출력하는 문제로 이스케이프해야 하는 문자가 섞여 있음에 주의해야 한다.

---

## 2. 코드

### 1. 구현 [Java]

역슬래시(`\`)와 큰 따옴표(`"`)에 대한 처리만 추가해줬다.

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("         ,r'\"7");
        System.out.println("r`-_   ,'  ,/");
        System.out.println(" \\. \". L_r'");
        System.out.println("   `~\\/");
        System.out.println("      |");
        System.out.println("      |");
    }
}
```

### 2. 구현 [C++]

역슬래시(`\`)와 큰 따옴표(`"`)에 대한 처리만 추가해줬다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cout << "         ,r'\"7\n";
    cout << "r`-_   ,'  ,/\n";
    cout << " \\. \". L_r'\n";
    cout << "   `~\\/\n";
    cout << "      |\n";
    cout << "      |\n";
}
```
