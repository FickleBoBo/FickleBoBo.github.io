## 1. 문제 풀이

주어진 정수를 소인수분해하는 문제로 소인수분해는 $2$ 이상의 소수로 주어진 수를 반복적으로 나누어서 $1$ 이 될 때까지 반복하면 된다. 이때 반복적으로 나누는 과정 덕분에 합성수로 나누게 되어 잘못 소인수분해를 할 일은 없는데 해당 합성수를 구성하는 소수로 이전에 이미 나누었을 것이기 때문이다. 따라서 나누는 수를 $1$ 씩 증가시키며 나누어도 소수로만 나눌 수 있다.

---

## 2. 코드

### 1. 소인수분해 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();

        int N = Integer.parseInt(br.readLine());
        int div = 2;

        while (N > 1) {
            while (N % div == 0) {
                N /= div;
                sb.append(div).append("\n");
            }

            div++;
        }

        System.out.println(sb);
    }
}
```

### 2. 소인수분해 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    int div = 2;
    while (n > 1) {
        while (n % div == 0) {
            n /= div;
            cout << div << '\n';
        }

        div++;
    }
}
```
