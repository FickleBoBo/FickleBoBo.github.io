## 1. 문제 풀이

<br>

$x$, $y$ 의 부호로 위치한 사분면을 판단할 수 있어서 조건문을 활용하면 해결할 수 있다.

- 1사분면의 경우 $x > 0$, $y > 0$
- 2사분면의 경우 $x < 0$, $y > 0$
- 3사분면의 경우 $x < 0$, $y < 0$
- 4사분면의 경우 $x > 0$, $y < 0$

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int x = Integer.parseInt(br.readLine());
        int y = Integer.parseInt(br.readLine());

        if (x > 0) {
            if (y > 0) {
                System.out.println(1);
            } else {
                System.out.println(4);
            }
        } else {
            if (y > 0) {
                System.out.println(2);
            } else {
                System.out.println(3);
            }
        }
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>

        if (y > 0) {
            cout << 1;
        } else {
            cout << 4;
        }
    } else {
        if (y > 0) {
            cout << 2;
        } else {
            cout << 3;
        }
    }
}
```
