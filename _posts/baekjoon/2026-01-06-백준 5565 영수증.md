## 1. 문제 풀이

<br>

$10$ 권의 가격 총합에서 $9$ 권의 가격을 빼면 남은 한 권의 가격을 구할 수 있다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int sum = Integer.parseInt(br.readLine());
        for (int i = 1; i < 10; i++) {
            sum -= Integer.parseInt(br.readLine());
        }

        System.out.println(sum);
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int sum;
    cin >> sum;

    for (int i = 1; i < 10; i++) {
        int x;
        cin >> x;
        sum -= x;
    }

    cout << sum;
}
```
