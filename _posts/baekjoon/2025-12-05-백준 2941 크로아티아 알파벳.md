## 1. 문제 풀이

<br>

주어진 문자열에서 크로아티아 알파벳을 구분해서 알파벳의 개수를 세야 하는 문제다. 크로아티아 알파벳을 찾으면 현재 바라보는 인덱스를 이동시키는 방식으로 크로아티아 알파벳의 개수를 세줬다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        char[] arr = br.readLine().toCharArray();
        int cnt = 0;

        for (int i = 0; i < arr.length; i++) {
            if (arr[i] == 'c') {
                if (i + 1 < arr.length && (arr[i + 1] == '=' || arr[i + 1] == '-')) {
                    i++;
                }
            } else if (arr[i] == 'd') {
                if (i + 1 < arr.length && arr[i + 1] == '-') {
                    i++;
                } else if (i + 2 < arr.length && arr[i + 1] == 'z' && arr[i + 2] == '=') {
                    i += 2;
                }
            } else if (arr[i] == 'l' || arr[i] == 'n') {
                if (i + 1 < arr.length && arr[i + 1] == 'j') {
                    i++;
                }
            } else if (arr[i] == 's' || arr[i] == 'z') {
                if (i + 1 < arr.length && arr[i + 1] == '=') {
                    i++;
                }
            }

            cnt++;
        }

        System.out.println(cnt);
    }
}
```
