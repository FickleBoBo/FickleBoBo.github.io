## 1. 문제 풀이

<br>

알파벳 소문자들로 주어진 문자열의 각 문자를 대문자로 출력하면 되는 문제로 내장 함수를 활용하거나 아스키 코드 자릿수를 활용하면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        String str = br.readLine();
        System.out.println(str.toUpperCase());
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;

    for (char c : s) {
        cout << (char)(c - 'a' + 'A');
    }
}
```
