## 1. 문제 풀이

여러 특수문자로 이루어진 고양이를 출력하는 문제로 이스케이프해야 하는 문자가 섞여 있음에 주의해야 한다.

---

## 2. 코드

### 1. 구현 [Java]

역슬래시(`\`)에 대한 처리만 추가해줬다.

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```

### 2. 구현 [C++]

역슬래시(`\`)에 대한 처리만 추가해줬다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cout << "\\    /\\" << '\n';
    cout << " )  ( ')" << '\n';
    cout << "(  /  )" << '\n';
    cout << " \\(__)|" << '\n';
}
```
