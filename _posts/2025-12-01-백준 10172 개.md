## 1. 문제 풀이

<br>

여러 특수문자로 이루어진 개를 출력하는 문제로 이스케이프해야 하는 문자가 섞여 있음에 주의해야 한다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

개를 이루는 문자 중 이스케이프 처리가 필요한 문자로 역슬래시(`\`)와 큰 따옴표(`"`)가 있어서 앞에 역슬래시를 붙여 이스케이프 처리만 추가해줬다.

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("|\\_/|");
        System.out.println("|q p|   /}");
        System.out.println("( 0 )\"\"\"\\");
        System.out.println("|\"^\"`    |");
        System.out.println("||_/=\\\\__|");
    }
}
```

<br>

### 2. 풀이 [C++]

개를 이루는 문자 중 이스케이프 처리가 필요한 문자로 역슬래시(`\`)와 큰 따옴표(`"`)가 있어서 앞에 역슬래시를 붙여 이스케이프 처리만 추가해줬다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cout << "|\\_/|" << '\n';
    cout << "|q p|   /}" << '\n';
    cout << "( 0 )\"\"\"\\" << '\n';
    cout << "|\"^\"`    |" << '\n';
    cout << "||_/=\\\\__|" << '\n';
}
```
