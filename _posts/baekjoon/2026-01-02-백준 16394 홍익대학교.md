## 1. 아이디어

<br>

1,946년이 개교 0주년이므로 주어진 N에서 1,946을 빼주면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        System.out.println(n - 1946);
    }
}
```

<br>

#include <bits/stdc++.h>

}
