## 1. 문제 풀이

<br>

주어진 $7$ 개의 수들 중 짝수의 합과 최솟값을 찾는 문제로 짝수는 $2$ 로 나눈 나머지가 $0$ 인지로 판별할 수 있다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        StringTokenizer st;

        int T = Integer.parseInt(br.readLine());
        for (int tc = 1; tc <= T; tc++) {
            int sum = 0;
            int min = 100;

            st = new StringTokenizer(br.readLine());
            for (int i = 0; i < 7; i++) {
                int x = Integer.parseInt(st.nextToken());

                if (x % 2 == 0) {
                    sum += x;
                    min = Math.min(min, x);
                }
            }

            sb.append(sum).append(" ").append(min).append("\n");
        }

        System.out.print(sb);
    }
}
```

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;

    for (int tc = 1; tc <= t; tc++) {
        int sum = 0;
        int mn = 100;

        for (int i = 0; i < 7; i++) {
            int x;
            cin >> x;

            if (x % 2 == 0) {
                sum += x;
                mn = min(mn, x);
            }
        }

        cout << sum << ' ' << mn << '\n';
    }
}
```
