## 1. 문제 풀이

전형적인 [LCS](/posts/longest-common-subsequence) 알고리즘 문제로 2차원 dp 테이블을 활용해서 구현했다.

문자열 $A$, $B$ 의 LCS를 구하는 dp 점화식은 아래와 같다.

<br>

$$
dp[i][j] =
\begin{cases}
0, & \text{if } i = 0 \text{ or } j = 0 \\\\
dp[i-1][j-1] + 1, & \text{if } A[i] = B[j] \\\\
\max(dp[i-1][j],\ dp[i][j-1]), & \text{otherwise}
\end{cases}
$$

---

## 2. 코드

### 1. 풀이 [Java]

dp 테이블 앞에 패딩을 한 칸씩 줬다.

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        char[] str1 = br.readLine().toCharArray();
        char[] str2 = br.readLine().toCharArray();
        int N = str1.length;
        int M = str2.length;

        int[][] dp = new int[1 + N][1 + M];
        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= M; j++) {
                if (str1[i - 1] == str2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        System.out.println(dp[N][M]);
    }
}
```

### 2. 풀이 [C++]

dp 테이블 앞에 패딩을 한 칸씩 줬다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s1, s2;
    cin >> s1 >> s2;

    int n = s1.size();
    int m = s2.size();

    vector<vector<int>> dp(1 + n, vector<int>(1 + m));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            if (s1[i - 1] == s2[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }

    cout << dp[n][m];
}
```
