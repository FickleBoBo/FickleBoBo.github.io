## 1. 아이디어

<br>

3킬로그램 봉지와 5킬로그램 봉지로 N킬로그램을 맞춰야 하는 문제로 사용한 3킬로그램 봉지가 5개를 넘어가면 그 5개 대신 5킬로그램 봉지 3개를 써서 더 적은 수의 봉지로 N킬로그램을 맞출 수 있다. 따라서 3킬로그램 봉지를 하나도 안썼을 때부터 4개를 썼을 때까지 중 N킬로그램을 맞출 수 있는 경우가 존재하는지 찾으면 된다.

- 3킬로그램 봉지를 아예 안썼을 때는 N을 5로 나눈 나머지가 0이면 N킬로그램을 맞출 수 있는 것이다.
- 3킬로그램 봉지를 한 개 썼을 때는 N - 3을 5로 나눈 나머지가 0이면 N킬로그램을 맞출 수 있는 것이다.
- 3킬로그램 봉지를 두 개 썼을 때는 N - 6을 5로 나눈 나머지가 0이면 N킬로그램을 맞출 수 있는 것이다.
- 3킬로그램 봉지를 세 개 썼을 때는 N - 9을 5로 나눈 나머지가 0이면 N킬로그램을 맞출 수 있는 것이다.
- 3킬로그램 봉지를 네 개 썼을 때는 N - 12을 5로 나눈 나머지가 0이면 N킬로그램을 맞출 수 있는 것이다.
- 위 경우 모두 N킬로그램을 맞추지 못하면 3킬로그램 봉지와 5킬로그램 봉지로 N킬로그램을 맞출 수 없는 것이다.

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int n = Integer.parseInt(br.readLine());

        if (n >= 0 && n % 5 == 0) {
            System.out.println(n / 5);
        } else if (n >= 3 && (n - 3) % 5 == 0) {
            System.out.println((n - 3) / 5 + 1);
        } else if (n >= 6 && (n - 6) % 5 == 0) {
            System.out.println((n - 6) / 5 + 2);
        } else if (n >= 9 && (n - 9) % 5 == 0) {
            System.out.println((n - 9) / 5 + 3);
        } else if (n >= 12 && (n - 12) % 5 == 0) {
            System.out.println((n - 12) / 5 + 4);
        } else {
            System.out.println(-1);
        }
    }
}
```
