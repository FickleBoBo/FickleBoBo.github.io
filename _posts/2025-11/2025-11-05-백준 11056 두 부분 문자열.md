## 문제 풀이

`A`와 `B`를 부분 문자열로 갖는 `S`의 최소 길이를 찾아야 하는 문제다. 해당 문제는 LCS를 활용하면 해결할 수 있는데 `A`와 `B`의 LCS의 길이를 `len`이라 할 때, `A`의 길이 + `B`의 길이 - `len`으로 `S`의 길이를 구할 수 있다.

`A`에 `B`의 문자를 기존 순서를 유지하며 삽입해서 `S`를 만든다고 했을 때 `S`는 `A`도 포함하고 `B`도 포함하게 된다. 이때 `A`에 존재하는 문자는 `B`에서 삽입을 안해도 되며 삽입을 안해도 되는 문자의 최대 수가 `A`와 `B`의 LCS의 길이만큼이 된다.

---

## 코드

### 1. Bottom-Up dp [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        char[] str1 = br.readLine().toCharArray();
        char[] str2 = br.readLine().toCharArray();
        int N = str1.length;
        int M = str2.length;

        int[][] dp = new int[1 + N][1 + M];
        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= M; j++) {
                if (str1[i - 1] == str2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        System.out.println(N + M - dp[N][M]);
    }
}
```
