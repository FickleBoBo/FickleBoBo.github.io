## 1. 문제 풀이

주어진 $N$ 개의 숫자에 대해 각 자릿수의 합을 구하는 문제로, 주어진 숫자들을 문자열로 입력 받은 후 각 자리의 수를 더하는 방식으로 해결하면 된다.

---

## 2. 코드

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        br.readLine();
        String str = br.readLine();

        int sum = 0;
        for (char c : str.toCharArray()) {
            sum += c - '0';
        }

        System.out.println(sum);
    }
}
```

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    string s;
    cin >> n >> s;

    int sum = 0;
    for (char c : s) {
        sum += c - '0';
    }

    cout << sum;
}
```
