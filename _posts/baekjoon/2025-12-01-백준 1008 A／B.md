## 1. 문제 풀이

<br>

두 정수 $A$, $B$ 의 나눗셈을 산술 연산자를 활용하여 구하면 된다. 이때 정수 타입간 나눗셈 연산은 몫만 구하는 연산이 될 수 있어서 주의해야 한다. 또한 실제 정답과의 오차가 $10^{-9}$ 이하여야 하는 점에 주의해야 한다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

`double` 형 변수와 `int` 형의 변수와의 연산은 `int` 형 변수가 `double` 형으로 형변환되어 연산이 수행되기에 `int` 형 $A$ 를 `double` 형으로 명시적 형변환해서 소수점 아래까지 나눗셈 연산이 잘 될 수 있도록 했다.

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int A = Integer.parseInt(st.nextToken());
        int B = Integer.parseInt(st.nextToken());

        System.out.println((double) A / B);
    }
}
```

<br>

### 2. 풀이 [Java]

처음부터 `double` 형으로 입력을 받아도 상관없다.

```java
import java.io.*;
```

### 3. 풀이 [C++]

`cout`은 소수점 아래 6자리까지만 출력하는 것이 기본값이기 때문에 `fixed`와 `setprecision(9)`을 통해 9자리까지 출력하도록 해줬다. 나눗셈의 경우 `double` 형 변수와 `int` 형의 변수와의 연산은 `int` 형 변수가 `double` 형으로 형변환되어 연산이 수행되기에 `int` 형 $A$ 를 `double` 형으로 명시적 형변환해서 소수점 아래까지 나눗셈 연산이 잘 될 수 있도록 했다.

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int a, b;
    cin >> a >> b;
    cout << fixed << setprecision(9) << (double)a / b;
}
```
