## 문제 풀이

[백준 11053번 - 가장 긴 증가하는 부분 수열]({% post_url 2025-11/2025-11-27-백준 11053 가장 긴 증가하는 부분 수열 %}) 문제에서 실제 LIS까지 출력해야 하는 문제다. dp 배열이 해당 부분 수열까지의 LIS의 길이를 담고 있으므로 이를 역순으로 조회하며 LIS의 길이와 일치하는 인덱스의 값이 실제 LIS의 원소가 되며 다시 길이를 1 감소시키고 다음 원소를 찾는 과정을 반복하면 된다.

---

## 코드

### 1. 2중 for문 LIS [Java]

역순으로 조회하므로 `Deque`를 스택으로 활용하여 정순으로 출력하게 해줬다.

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        StringBuilder sb = new StringBuilder();
        StringTokenizer st;

        int N = Integer.parseInt(br.readLine());

        int[] arr = new int[N];
        st = new StringTokenizer(br.readLine());
        for (int i = 0; i < N; i++) {
            arr[i] = Integer.parseInt(st.nextToken());
        }

        int[] dp = new int[N];
        Arrays.fill(dp, 1);
        int max = 0;

        for (int i = 0; i < N; i++) {
            for (int j = 0; j < i; j++) {
                if (arr[i] > arr[j]) dp[i] = Math.max(dp[i], dp[j] + 1);
            }

            max = Math.max(max, dp[i]);
        }

        sb.append(max).append('\n');

        Deque<Integer> stack = new ArrayDeque<>();
        int tmp = max;
        for (int i = N - 1; i >= 0; i--) {
            if (dp[i] == tmp) {
                stack.push(arr[i]);
                tmp--;
            }
        }

        while (!stack.isEmpty()) {
            sb.append(stack.pop()).append(" ");
        }

        bw.write(sb.toString());
        bw.flush();
    }
}
```
