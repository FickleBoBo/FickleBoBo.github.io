## 1. 문제 풀이

<br>

나무 조각의 위치 변경 과정이 버블 정렬 과정과 동일하다. 따라서 버블 정렬을 구현 후 스왑이 발생할 때 현재 나무 조각의 수들을 출력해줬다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        StringTokenizer st = new StringTokenizer(br.readLine());

        int[] arr = new int[5];
        for (int i = 0; i < 5; i++) {
            arr[i] = Integer.parseInt(st.nextToken());
        }

        for (int i = 0; i < 4; i++) {
            for (int j = 0; j < 4 - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    int tmp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = tmp;

                    for (int n : arr) {
                        sb.append(n).append(" ");
                    }
                    sb.append("\n");
                }
            }
        }

        System.out.println(sb);
    }
}
```

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    vector<int> v(5);
    for (int& x : v) cin >> x;

    for (int i = 0; i < 4; i++) {
        for (int j = 0; j < 4 - i; j++) {
            if (v[j] > v[j + 1]) {
                swap(v[j], v[j + 1]);

                for (int x : v) {
                    cout << x << ' ';
                }
                cout << '\n';
            }
        }
    }
}
```
