## 1. 문제 풀이

<br>

$1,946$ 년이 개교 $0$ 주년이므로 주어진 $N$ 에서 $1,946$ 을 빼주면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int N = sc.nextInt();
        System.out.println(N - 1946);
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    cout << n - 1946;
}
```
