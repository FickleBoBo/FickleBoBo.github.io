## 1. 문제 풀이

<br>

오늘 날짜를 출력하면 된다. 서버 시간대가 UTC여서 한국 시간과 9시간 차이가 나는 점만 주의하면 된다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("2025-12-30");
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

}
```
