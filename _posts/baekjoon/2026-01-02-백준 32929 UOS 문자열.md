## 1. 문제 풀이

<br>

`UOS`가 반복되는 문자열에서 $x$ 번째 문자를 구하는 문제로 세 문자 단위로 같은 문자가 반복되므로 $x$ 를 $3$ 으로 나눈 나머지로 판단할 수 있다. 문자열에서 첫 번째 문자의 인덱스가 $0$ 이므로 $x - 1$ 번 인덱스의 문자를 구해야하고 이는 $x - 1$ 을 $3$ 으로 나눈 나머지와 같다.

---

## 2. 코드

<br>

### 1. 풀이 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int x = Integer.parseInt(br.readLine());

        System.out.println("UOS".charAt((x - 1) % 3));
    }
}
```

<br>

### 2. 풀이 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int x;
    cin >> x;
    cout << "UOS"[(x - 1) % 3];
}
```
