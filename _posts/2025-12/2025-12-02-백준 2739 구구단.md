## 1. 문제 풀이

구구단의 $N$ 단을 출력하면 되는 문제로 반복문을 활용하면 해결할 수 있다.

---

## 2. 코드

### 1. 구현 [Java]

```java
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());

        for (int i = 1; i <= 9; i++) {
            System.out.printf("%d * %d = %d\n", N, i, N * i);
        }
    }
}
```

### 2. 구현 [C++]

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N;
    cin >> N;

    for (int i = 1; i <= 9; i++) {
        cout << N << " * " << i << " = " << N * i << '\n';
    }
}
```
